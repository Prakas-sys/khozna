# Khozna Brand Website - #1 Direct Rental Platform in Nepal 🇳🇵🚀

Khozna is a premium property rental platform tailored for the Nepal market, featuring direct owner-to-tenant communication, verified listings, and integrated security features.

This repository contains the official brand website for Khozna, built with a focus on trust, mission, and advanced visual storytelling.

## 🌟 Features

- **#1 Direct Rental Branding:** Clearly communicates the platform's core value: direct owner-tenant connection without middlemen.
- **Premium Design:** Implements the "Zen Antique Soft" aesthetic with custom typography and a sky-blue/white color palette.
- **Advanced Animations:** Utilizes Framer Motion for 3D parallax effects, interactive elements, and smooth entrance transitions.
- **Traditional Branding:** Integrates high-quality visuals of traditional Nepali rental spaces.
- **App Teaser:** Includes "App Coming Soon" messaging and visuals for mobile platforms.

## 🛠️ Technology Stack

- **Framework:** React + Vite
- **Language:** TypeScript
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Typography:** Google Fonts (Lexend, Zen Antique Soft)

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Prakas-sys/Khozna.web.git
   cd Khozna.web
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🛡️ Security & Privacy

- **Direct Flows:** Khozna prioritizes direct communication.
- **Verification:** Features "Verified Owner Listed" badges to build trust.
- **Production Safety:** Ensure all sensitive API keys (when added) are managed via `.env` files (already ignored by `.gitignore`).
