# 🚀 GSD (Get Shit Done) - Khozna Master Control
Control and manage the GSD autonomous agent loop for the Khozna project.

## /gsd:status
Show the current project status and the technical roadmap.
1. Read `GSD.md` to identify current mission.
2. Read `progress.txt` to display the audit trail.
3. Summarize the next 3 tasks for the user.

## /gsd:execute
Initiate the autonomous agent loop to finish the current tasks.
// turbo
1. Read the first [ ] task in `GSD.md`.
2. Check `progress.txt` for the current context.
3. Perform the technical task (code, fix, refactor).
4. Update `GSD.md` (mark task as [x]).
5. Log progress in `progress.txt`.
6. Output: "GSD Loop Complete. Mission: [Task Name] Finished! 🎉"

## /gsd:plan
Create or update the technical roadmap based on the current vision.
1. Read `PRD.md` to identify the vision and roadmaps.
2. Generate a technical breakdown in `GSD.md`.
3. Ask the user for confirmation! 🛋️✨🕯️🕺
