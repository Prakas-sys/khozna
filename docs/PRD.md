# 🇳🇵 Khozna - Product Requirements Document (PRD)
Created by the USER (Lead Designer) & Antigravity (Implementation Partner)

## 1. Project Vision
Khozna is a premium, secure, and user-friendly property rental platform specifically designed for the Nepal market. It connects owners directly with tenants, focusing on **Verified Listings**, **Trust**, and **Premium UI/UX**.

## 2. Core Features (Visualized by Designer)
- **Direct Communication:** Owners and tenants chatting in real-time.
- **Verified Identity (KYC):** Integration with citizenship documents and location markers to build trust.
- **Smooth Search:** Fast, intuitive, and "smooth" property discovery.
- **Property Management:** Easy 5-step property listing flow with AI assistance.

## 3. Design Principles (DO NOT BREAK)
- **Aesthetics:** High-end, clean, and vibrant.
- **Feel:** "Smoothn" - use haptics, transitions, and polished micro-animations.
- **Branding:** Use Khozna Brand Colors (`#00A3E1`) and premium fonts (Outfit, Playfair Display).
